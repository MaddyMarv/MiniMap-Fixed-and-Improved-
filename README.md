### [Minimap Fixed and Improved](https://www.nexusmods.com/warhammer40kdarktide/mods/633)
<img width="1689" height="476" alt="image" src="https://github.com/user-attachments/assets/1e5fcc37-66cf-463c-bf92-b1b7dfe93dd8" />

Standalone mod that gives you an actual working minimap on your screen. It is fully customizable so you can tweak the size, zoom level, and opacity. It tracks teammates and objectives so it makes it a lot easier to stay grouped up during fights and not get lost.

NOW SHOWS STRIKEMAP. If you install Strike Map mod, you can see the real mission map live! Please when you install it, disable their minimap in settings!
